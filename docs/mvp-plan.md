# haill 农业平台 MVP 计划

## 定位

农业技术 + 问题求助 + 本地供需撮合平台，先解决“不会种、不会养、出问题没人管、东西卖不出去”。

## P0 功能

1. 技术教程：分类、列表、详情、老师发布、后台审核。
2. 问题求助：农户图文/视频求助，老师回复。
3. 老师入驻：提交资料，后台审核。
4. 农户货源：产品、数量、地区、上市时间、图片、视频、自送价、上门收购价。
5. 收购需求：产品、数量、地区、收购时间、自送参考价、上门收购参考价。
6. 后台审核：老师、教程、问题、货源、需求。

## 暂不做

在线支付、物流、直播、农资商城、金融保险、复杂地图匹配、AI 病虫害识别。

## 关键风控

- 联系前提示：不要提前支付保证金、押金、运费。
- 关键词拦截：保证金、押金、先打款、加微信、二维码、稳赚、神药。
- 首次发布内容进入审核。
- 视频验货保留上传时间、发布人、绑定货源。
